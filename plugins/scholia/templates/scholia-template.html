Try the commenting

This is the template — replace this section when generating a real document.

Hover any line. A + button appears in the left gutter — click it to comment on the whole line. Or select any text inside a line and a "+ Add comment" toolbar pops up for a finer-grained comment.

Use the "Copy comments" button in the top right to put a formatted block on your clipboard, then paste it back into chat for Claude to address.

  • List items are commentable line by line.
  • So are paragraphs, headings, and code lines.
  • Comments persist in browser localStorage keyed by the file name.
